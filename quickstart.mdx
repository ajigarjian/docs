# Quick Start Guide
1. Create an account and get your API key at [carbonmkts.com](https://carbonmkts.com)
2. Add your API key to the request header as `Authorization: Api-Key YOUR_API_KEY`

```python
headers = {
    'Authorization': 'Api-Key YOUR_API_KEY'
}
```

3. Make API requests using our endpoints

## API Examples

<CodeGroup>

```python Registry Data
import requests

headers = {
    'Authorization': 'Api-Key YOUR_API_KEY'
}

# Get all projects from a specific registry
params = {
    'registry': 'verra',
    'limit': 100
}

response = requests.get(
    'https://api.carbonmkts.com/registries',
    headers=headers,
    params=params
)

registry_data = response.json()
```

```python Project Search
import requests

headers = {
    'Authorization': 'Api-Key YOUR_API_KEY'
}

# Search for forestry projects in Brazil
params = {
    'project_type': 'forestry',
    'country': 'brazil',
    'status': 'active',
    'limit': 50
}

response = requests.get(
    'https://api.carbonmkts.com/projects',
    headers=headers,
    params=params
)

projects = response.json()
```

```python Transaction Data
import requests

headers = {
    'Authorization': 'Api-Key YOUR_API_KEY'
}

# Get recent retirements
params = {
    'transaction_type': 'retirement',
    'page_size': 50
}

response = requests.get(
    'https://api.carbonmkts.com/transactions',
    headers=headers,
    params=params
)

transactions = response.json()
```

</CodeGroup>

## Rate Limits

- Free tier: 100 requests per hour
- Basic tier: 1,000 requests per hour
- Premium tier: 10,000 requests per hour

For higher rate limits, please contact [support@carbonmkts.com](mailto:support@carbonmkts.com).
